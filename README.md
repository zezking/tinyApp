## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies

```javascript
npm install
```

- Run the development web server using the

```javascript
node express_server.js
```

- visit the following link

```
localhost:8080/urls;
```
